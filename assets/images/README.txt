Put your real images in this folder (assets/images/) using these exact filenames.
The website already looks for them — nothing else needs to change.

logo.png            -> Header + footer logo. Square image works best, e.g. 128x128px, PNG with transparent background.

hero-1.png          -> Main hero phone screenshot (front, center phone).      Recommended size: 750x1500px (or any 9:19 portrait ratio)
hero-2.png          -> Hero phone screenshot (right/side phone).              Recommended size: 750x1500px
hero-3.png          -> Hero phone screenshot (left/back phone).               Recommended size: 750x1500px

screen-browse.png   -> "Browse by category" screenshot in the Preview section. Recommended size: 750x1500px
screen-details.png  -> "Rich listing details" screenshot in the Preview section. Recommended size: 750x1500px
screen-chat.png     -> "Direct, secure chat" screenshot in the Preview section.  Recommended size: 750x1500px

Notes:
- Images are cropped to fill the phone screen (object-fit: cover), so a portrait
  phone-app screenshot (roughly 9:19 or 9:16) will look best — no need to match exactly.
- If a file is missing or misnamed, the page automatically falls back to the
  gradient placeholder — nothing will break.
- PNG or JPG both work. Keep each file under ~500KB for fast loading.
